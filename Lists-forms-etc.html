1. How do we define lists?
a. dl Stands for "definition list" and acts as a container for the whole list.
b. dt Stands for "definition term" and defines the term.
c. dd Stands for "definition description" and provides the explanation.

HTML Tables

1. Tables have a head, body, foot, rows, and caption.
1. Table: The container element that defines the table.
2. Table Row: Represents a single row within the table. You need at least one for a table.
3. Table Header: Defines a header cell within a row. These cells are bold and centered by default, typically used for headings.
4. Table Data: Represents standard data cells within a table. These are the "regular" cells in each row.
5. Caption: Provides a title or description for the table.
6. Thead: Groups the header rows of a table. Typically contains and elements.
7. Tbody: Groups the main body rows of a table. Contains and elements.
8. Tfoot: Groups the footer rows of a table, often used for summary information or totals.
2. How to use the mentioned componets of tables.
1. All code of a table is in an angled brackets with the word table inside.
2. For the head we use thead
3. For the body we use tbody
4. For the foot we use tfoot
5. For the rows we use tr
6. For the header we use th
7. For the data we use td
8. For the caption we use caption

HTML Forms

How to use forms in HTML
We initiate form using the keyword form
Then we specify the action and method
Then we use a lable to collect info from the user
Then we use input to collect the info
The we use id. An id is a unique identifier.
Then we use required to let the user know they have to provide that info
There are different kinds of information we can collect from the user using forms.
For example, we can collect a string type of info, a number, a date, an email, a password, a radio button, a checkbox, a select option, a text area, a button, a submit button, and a reset button.
1. For a string type of info we use input type="text"
2. For a number we use input type="number"
3. For a date we use input type="date"
4. For an email we use input type="email"
5. For a password we use input type="password"
6. For a radio button we use input type="radio"
7. For a checkbox we use input type="checkbox"
8. For a select option we use select
9. For a text area we use textarea
10. For a button we use button
11. For a submit button we use input type="submit"
12. For a reset button we use input type="reset"
Here are a couple of examples.
1. For a radio button: input type="radio" id="gender" name="Gender" value="Female"
2. For a checkbox: input type="checkbox" name="Hobbies" id="Sports" value="Sports"
3. For a select option: select id="country" name="country"
4. For a text area: textarea id="Description" name="Description" rows="5" cols="30"
